# 名字标签

## Introduction @unplugged

我们使用图像编辑器创建一个很酷的名称标签！

## 第 1 步

我们将模仿顶部和底部带有红条的名称标签。
添加``||scene:设置背景颜色为||`` 程序块，将屏幕涂成红色。

```blocks
// @highlight
scene.setBackgroundColor(2)
```
